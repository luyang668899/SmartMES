# MES 系统用户手册

## 1. 系统概述

MES（Manufacturing Execution System）是一套面向制造业的生产执行系统，旨在提高生产效率、优化生产流程、确保产品质量。本系统采用插件架构，支持功能的扩展和定制，可适应不同规模和类型的制造企业。

### 1.1 系统功能

- **基础管理**：用户管理、角色权限、基础数据维护
- **订单管理**：订单创建、跟踪、执行
- **生产管理**：生产计划、调度、执行
- **质量管理**：质量控制、检验、统计分析
- **供应链管理**：供应商管理、采购管理、库存管理
- **设备管理**：设备连接、监控、维护
- **数据分析**：KPI 指标、报表、决策支持
- **系统集成**：与 ERP、CRM 等系统集成
- **人工智能与机器学习**：预测分析、异常检测、质量预测
- **移动应用**：现场数据采集、实时通知、移动设备管理
- **能源管理**：能耗监控、能源分析、成本优化
- **预测性维护**：设备健康监测、故障预测、维护计划

### 1.2 系统架构

- **前端**：基于 Web 的用户界面
- **后端**：Java 开发的服务端应用
- **数据库**：MySQL
- **中间件**：Tomcat

## 2. 系统安装

### 2.1 硬件要求

- **服务器**：至少 8GB 内存，500GB 硬盘，4 核 CPU
- **客户端**：至少 4GB 内存，现代浏览器（Chrome、Firefox、Edge）

### 2.2 软件要求

- **操作系统**：Windows Server 2012 或更高版本，Linux（Ubuntu 18.04 或更高版本），macOS 10.14 或更高版本
- **Java**：JDK 25 或更高版本
- **数据库**：MySQL 5.7 或 8.0
- **应用服务器**：Tomcat 8.5 或 9.0

### 2.3 安装步骤

1. **安装 Java**
   - 下载并安装 JDK 25 或更高版本
   - 配置环境变量

2. **安装 MySQL**
   - 下载并安装 MySQL 5.7 或 8.0
   - 创建数据库和用户
   - 导入初始数据

3. **安装 Tomcat**
   - 下载并安装 Tomcat 8.5 或 9.0
   - 配置 Tomcat 环境

4. **部署 MES 系统**
   - 将 MES 系统的 WAR 文件复制到 Tomcat 的 webapps 目录
   - 启动 Tomcat 服务
   - 访问系统 URL：http://localhost:8080/mes

5. **配置系统参数**
   - 登录系统（默认用户名：admin，密码：admin）
   - 配置系统参数，如数据库连接、邮件服务器等

### 5.3 数据问题

- **数据丢失**：检查数据备份、联系系统管理员

### 5.4 系统问题

- **系统卡顿**：检查系统负载、清理浏览器缓存
- **功能不可用**：检查权限设置、系统配置
- **系统崩溃**：联系系统管理员、查看系统日志
