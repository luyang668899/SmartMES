# SmartMES 详细使用说明书

## 1. 系统概述

SmartMES 是一套基于 qcadoo MES 构建的制造执行系统，专为中小型制造企业设计，旨在提高生产效率、优化生产流程、确保产品质量。系统采用插件架构，支持功能扩展和定制，可适应不同规模和类型的制造企业。

### 1.1 系统架构

- **前端**：基于 Web 的用户界面，支持现代浏览器
- **后端**：Java 开发的服务端应用，基于 Spring 框架
- **数据库**：MySQL 关系型数据库
- **中间件**：Tomcat 应用服务器
- **插件系统**：模块化设计，支持功能扩展

### 1.2 核心功能模块

| 模块 | 主要功能 | 适用角色 |
|------|----------|----------|
| **基础管理** | 公司、工厂、部门、员工、班次管理 | 系统管理员 |
| **订单管理** | 订单创建、跟踪、执行、分析 | 销售、生产计划 |
| **生产管理** | 生产计划、调度、执行、监控 | 生产主管、车间主任 |
| **质量管理** | 质量控制、检验、统计分析 | 质量主管、检验员 |
| **供应链管理** | 供应商、采购、库存管理 | 采购、仓库 |
| **设备管理** | 设备连接、监控、维护 | 设备管理员 |
| **数据分析** | KPI 指标、报表、决策支持 | 管理层、分析师 |
| **系统集成** | 与 ERP、CRM 等系统集成 | IT 管理员 |
| **人工智能** | 预测分析、异常检测、质量预测 | 数据分析师 |
| **移动应用** | 现场数据采集、实时通知 | 一线工人、巡检员 |

## 2. 系统登录与导航

### 2.1 登录系统

1. **打开浏览器**：推荐使用 Chrome、Firefox 或 Edge 浏览器
2. **输入系统 URL**：如 `http://localhost:8080/mes`（根据实际部署地址调整）
3. **输入用户名和密码**：默认管理员账号为 `admin`，密码为 `admin`
4. **点击登录按钮**：进入系统主界面

### 2.2 系统主界面

系统主界面由以下部分组成：

- **顶部导航栏**：显示系统名称、用户信息、通知中心、系统设置
- **左侧菜单**：系统功能模块导航
- **主内容区**：当前功能模块的详细内容
- **底部状态栏**：显示系统版本、当前时间等信息

### 2.3 导航操作

- **模块切换**：点击左侧菜单中的模块名称
- **功能切换**：在模块内点击子菜单
- **返回上一级**：使用浏览器的返回按钮或面包屑导航
- **搜索功能**：使用顶部导航栏的搜索框
- **快速访问**：将常用功能添加到收藏夹

### 2.4 用户设置

1. **个人资料**：点击顶部导航栏的用户头像 → 个人资料
   - 修改个人信息
   - 修改密码
   - 设置语言偏好

2. **通知设置**：点击顶部导航栏的通知图标 → 通知设置
   - 配置通知类型
   - 设置通知接收方式

## 3. 基础管理

### 3.1 公司管理

1. **创建公司**
   - 点击左侧菜单 → 基础管理 → 公司管理 → 新增
   - 填写公司信息：名称、地址、联系人、电话等
   - 点击保存按钮

2. **编辑公司**
   - 在公司列表中选择公司 → 点击编辑按钮
   - 修改公司信息
   - 点击保存按钮

3. **删除公司**
   - 在公司列表中选择公司 → 点击删除按钮
   - 确认删除操作

### 3.2 工厂管理

1. **创建工厂**
   - 点击左侧菜单 → 基础管理 → 工厂管理 → 新增
   - 选择所属公司
   - 填写工厂信息：名称、地址、负责人等
   - 点击保存按钮

2. **编辑工厂**
   - 在工厂列表中选择工厂 → 点击编辑按钮
   - 修改工厂信息
   - 点击保存按钮

3. **删除工厂**
   - 在工厂列表中选择工厂 → 点击删除按钮
   - 确认删除操作

### 3.3 部门管理

1. **创建部门**
   - 点击左侧菜单 → 基础管理 → 部门管理 → 新增
   - 选择所属公司和工厂
   - 填写部门信息：名称、负责人、职责等
   - 点击保存按钮

2. **编辑部门**
   - 在部门列表中选择部门 → 点击编辑按钮
   - 修改部门信息
   - 点击保存按钮

3. **删除部门**
   - 在部门列表中选择部门 → 点击删除按钮
   - 确认删除操作

### 3.4 员工管理

1. **创建员工**
   - 点击左侧菜单 → 基础管理 → 员工管理 → 新增
   - 填写员工信息：姓名、工号、部门、职位、联系方式等
   - 点击保存按钮

2. **编辑员工**
   - 在员工列表中选择员工 → 点击编辑按钮
   - 修改员工信息
   - 点击保存按钮

3. **员工技能管理**
   - 在员工详情页面 → 点击技能标签
   - 点击添加技能按钮
   - 选择技能类型和级别
   - 点击保存按钮

### 3.5 班次管理

1. **创建班次**
   - 点击左侧菜单 → 基础管理 → 班次管理 → 新增
   - 填写班次信息：名称、开始时间、结束时间、休息时间等
   - 点击保存按钮

2. **编辑班次**
   - 在班次列表中选择班次 → 点击编辑按钮
   - 修改班次信息
   - 点击保存按钮

3. **删除班次**
   - 在班次列表中选择班次 → 点击删除按钮
   - 确认删除操作

## 4. 订单管理

### 4.1 订单创建

1. **进入订单管理**
   - 点击左侧菜单 → 订单管理 → 订单列表

2. **创建新订单**
   - 点击新增按钮
   - 填写订单基本信息：
     - 订单编号（系统自动生成或手动输入）
     - 客户信息
     - 订单类型
     - 优先级
     - 交付日期
   - 填写产品信息：
     - 选择产品
     - 输入数量
     - 选择单位
     - 填写交货期
   - 点击保存按钮

3. **订单审核**
   - 在订单列表中选择订单 → 点击审核按钮
   - 确认订单信息
   - 点击审核通过按钮

### 4.2 订单跟踪

1. **订单状态查询**
   - 在订单列表中查看订单状态
   - 使用筛选器按状态、日期、客户等条件筛选

2. **订单详情查看**
   - 在订单列表中选择订单 → 点击详情按钮
   - 查看订单基本信息、产品信息、生产进度、质量信息等

3. **生产进度跟踪**
   - 在订单详情页面 → 点击生产进度标签
   - 查看各工序完成情况
   - 查看生产计划与实际执行对比

4. **质量信息跟踪**
   - 在订单详情页面 → 点击质量信息标签
   - 查看质量检验结果
   - 查看不合格品信息

### 4.3 订单执行

1. **生产任务分配**
   - 在订单详情页面 → 点击生产任务标签
   - 点击分配任务按钮
   - 选择生产线、班次、员工
   - 点击保存按钮

2. **生产数据记录**
   - 一线员工通过移动应用或车间终端记录生产数据
   - 生产主管在系统中查看实时生产数据

3. **订单完成**
   - 当所有生产任务完成后，订单状态自动更新为完成
   - 点击订单详情页面的完成按钮确认订单完成

4. **订单交付**
   - 点击订单详情页面的交付按钮
   - 填写交付信息：交付日期、交付数量、交付方式等
   - 点击保存按钮

### 4.4 订单分析

1. **订单统计报表**
   - 点击左侧菜单 → 订单管理 → 订单报表
   - 选择报表类型：订单数量、订单金额、交付及时率等
   - 选择时间范围
   - 点击生成报表按钮

2. **订单趋势分析**
   - 点击左侧菜单 → 订单管理 → 订单趋势
   - 查看订单数量、金额的月度/季度/年度趋势
   - 分析订单变化原因

3. **客户订单分析**
   - 点击左侧菜单 → 订单管理 → 客户订单分析
   - 查看各客户的订单数量、金额、交付情况
   - 分析客户订单变化趋势

## 5. 生产管理

### 5.1 生产计划

1. **创建生产计划**
   - 点击左侧菜单 → 生产管理 → 生产计划 → 新增
   - 填写计划信息：计划名称、计划周期、负责人等
   - 点击保存按钮

2. **添加生产任务**
   - 在生产计划详情页面 → 点击添加任务按钮
   - 选择订单
   - 选择产品
   - 输入计划数量
   - 选择计划开始和结束时间
   - 点击保存按钮

3. **计划排产**
   - 点击左侧菜单 → 生产管理 → 生产排产
   - 查看生产任务甘特图
   - 拖拽任务调整开始和结束时间
   - 点击保存排产按钮

### 5.2 生产执行

1. **生产任务列表**
   - 点击左侧菜单 → 生产管理 → 生产任务
   - 查看待执行、执行中、已完成的任务
   - 使用筛选器按生产线、班次、状态等条件筛选

2. **开始生产任务**
   - 在生产任务列表中选择任务 → 点击开始按钮
   - 确认任务信息
   - 点击确认开始按钮

3. **记录生产数据**
   - 在生产任务详情页面 → 点击数据记录标签
   - 输入生产数量
   - 记录生产时间
   - 填写生产备注
   - 点击保存按钮

4. **暂停生产任务**
   - 在生产任务详情页面 → 点击暂停按钮
   - 选择暂停原因
   - 点击确认暂停按钮

5. **完成生产任务**
   - 在生产任务详情页面 → 点击完成按钮
   - 确认生产数据
   - 点击确认完成按钮

### 5.3 生产监控

1. **实时生产监控**
   - 点击左侧菜单 → 生产管理 → 生产监控
   - 查看各生产线的实时状态
   - 查看生产任务的实时执行情况
   - 查看设备运行状态

2. **生产异常报警**
   - 在生产监控页面查看异常报警信息
   - 点击异常详情查看具体信息
   - 点击处理按钮处理异常

3. **生产数据分析**
   - 点击左侧菜单 → 生产管理 → 生产分析
   - 查看生产效率、设备利用率、OEE 等指标
   - 分析生产瓶颈

## 6. 质量管理

### 6.1 质量检验计划

1. **创建检验计划**
   - 点击左侧菜单 → 质量管理 → 检验计划 → 新增
   - 填写计划信息：计划名称、检验类型、检验标准等
   - 点击保存按钮

2. **添加检验项目**
   - 在检验计划详情页面 → 点击添加项目按钮
   - 填写项目信息：项目名称、检验方法、判定标准等
   - 点击保存按钮

3. **分配检验任务**
   - 在检验计划详情页面 → 点击分配任务按钮
   - 选择检验员
   - 选择检验时间
   - 点击保存按钮

### 6.2 质量检验执行

1. **检验任务列表**
   - 点击左侧菜单 → 质量管理 → 检验任务
   - 查看待执行、执行中、已完成的检验任务
   - 使用筛选器按检验类型、状态、检验员等条件筛选

2. **执行检验任务**
   - 在检验任务列表中选择任务 → 点击执行按钮
   - 填写检验结果：
     - 输入实测值
     - 选择判定结果（合格/不合格）
     - 上传检验图片（可选）
     - 填写检验备注
   - 点击保存按钮

3. **不合格品处理**
   - 在检验任务详情页面 → 点击不合格品处理按钮
   - 选择处理方式：返工、报废、降级等
   - 填写处理备注
   - 点击保存按钮

### 6.3 质量分析

1. **质量统计报表**
   - 点击左侧菜单 → 质量管理 → 质量报表
   - 选择报表类型：合格率、不合格品数量、不良率等
   - 选择时间范围
   - 点击生成报表按钮

2. **SPC 分析**
   - 点击左侧菜单 → 质量管理 → SPC 分析
   - 选择检验项目
   - 选择时间范围
   - 查看控制图
   - 分析过程稳定性

3. **质量趋势分析**
   - 点击左侧菜单 → 质量管理 → 质量趋势
   - 查看质量指标的月度/季度/年度趋势
   - 分析质量变化原因

4. **供应商质量评估**
   - 点击左侧菜单 → 质量管理 → 供应商质量评估
   - 查看各供应商的质量表现
   - 分析供应商质量变化趋势

## 7. 供应链管理

### 7.1 供应商管理

1. **创建供应商**
   - 点击左侧菜单 → 供应链管理 → 供应商管理 → 新增
   - 填写供应商信息：名称、地址、联系人、电话等
   - 填写供应商评估信息：质量等级、交付能力、服务水平等
   - 点击保存按钮

2. **编辑供应商**
   - 在供应商列表中选择供应商 → 点击编辑按钮
   - 修改供应商信息
   - 点击保存按钮

3. **供应商评估**
   - 在供应商详情页面 → 点击评估标签
   - 点击添加评估按钮
   - 填写评估信息：质量、交付、服务、价格等
   - 点击保存按钮

### 7.2 采购管理

1. **创建采购请求**
   - 点击左侧菜单 → 供应链管理 → 采购管理 → 采购请求 → 新增
   - 填写请求信息：请求编号、需求部门、需求日期等
   - 填写物料信息：物料名称、规格、数量、需求日期等
   - 点击保存按钮

2. **采购请求审批**
   - 在采购请求列表中选择请求 → 点击审批按钮
   - 审核采购请求信息
   - 点击审批通过或拒绝按钮

3. **创建采购订单**
   - 在采购请求详情页面 → 点击生成采购订单按钮
   - 选择供应商
   - 填写订单信息：订单编号、交货日期、付款方式等
   - 点击保存按钮

4. **采购订单跟踪**
   - 点击左侧菜单 → 供应链管理 → 采购管理 → 采购订单
   - 查看采购订单状态
   - 在采购订单详情页面查看交付进度

### 7.3 库存管理

1. **入库管理**
   - 点击左侧菜单 → 供应链管理 → 库存管理 → 入库管理
   - 点击新增按钮
   - 选择入库类型：采购入库、生产入库、退货入库等
   - 填写入库信息：供应商、仓库、入库日期等
   - 填写物料信息：物料名称、规格、数量、批次等
   - 点击保存按钮

2. **出库管理**
   - 点击左侧菜单 → 供应链管理 → 库存管理 → 出库管理
   - 点击新增按钮
   - 选择出库类型：生产领料、销售出库、退货出库等
   - 填写出库信息：领料部门、仓库、出库日期等
   - 填写物料信息：物料名称、规格、数量、批次等
   - 点击保存按钮

3. **库存查询**
   - 点击左侧菜单 → 供应链管理 → 库存管理 → 库存查询
   - 输入物料名称或编码
   - 选择仓库
   - 点击查询按钮
   - 查看物料库存状态

4. **库存预警**
   - 点击左侧菜单 → 供应链管理 → 库存管理 → 库存预警
   - 查看库存低于安全库存的物料
   - 查看库存高于最大库存的物料

## 8. 设备管理

### 8.1 设备信息管理

1. **创建设备**
   - 点击左侧菜单 → 设备管理 → 设备信息 → 新增
   - 填写设备基本信息：设备编号、设备名称、设备类型、型号等
   - 填写设备技术参数：功率、产能、精度等
   - 填写设备位置信息：工厂、车间、生产线等
   - 点击保存按钮

2. **编辑设备**
   - 在设备列表中选择设备 → 点击编辑按钮
   - 修改设备信息
   - 点击保存按钮

3. **设备台账**
   - 点击左侧菜单 → 设备管理 → 设备台账
   - 查看设备基本信息、技术参数、维护记录等

### 8.2 设备监控

1. **实时监控**
   - 点击左侧菜单 → 设备管理 → 设备监控
   - 查看设备实时状态：运行、待机、故障、维护等
   - 查看设备实时参数：温度、压力、转速等
   - 查看设备报警信息

2. **历史数据查询**
   - 在设备详情页面 → 点击历史数据标签
   - 选择时间范围

## 14. 故障排除

### 14.1 登录问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无法登录系统 | 用户名或密码错误 | 确认用户名和密码是否正确，注意大小写 |
| 账号被锁定 | 多次登录失败 | 联系系统管理员解锁账号 |
| 无法访问系统 | 网络连接失败 | 检查网络连接，确认系统服务是否运行 |
| 登录超时 | 会话过期 | 重新登录系统 |

### 14.2 操作问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无法保存数据 | 必填字段未填写 | 检查所有必填字段是否填写完整 |
| 无法删除记录 | 有相关记录或无删除权限 | 检查是否有相关记录，确认是否有删除权限 |
| 操作超时 | 网络连接缓慢或系统负载高 | 检查网络连接，稍后重试 |
| 功能不可用 | 权限不足或功能未启用 | 联系系统管理员确认权限，检查功能是否启用 |

### 14.3 数据问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 数据丢失 | 未保存数据或系统故障 | 定期保存数据，联系系统管理员恢复数据 |
| 数据错误 | 输入错误或系统计算错误 | 检查数据输入，联系系统管理员修复错误 |
| 数据不一致 | 数据同步失败或并发操作 | 联系系统管理员检查数据同步，避免并发操作 |
| 数据重复 | 导入错误或手动输入错误 | 检查数据导入，删除重复数据 |

### 14.4 系统问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 系统卡顿 | 系统负载高或内存不足 | 检查系统资源使用情况，重启系统服务 |
| 页面加载缓慢 | 网络连接缓慢或页面数据量大 | 检查网络连接，优化页面数据加载 |
| 系统崩溃 | 内存溢出或系统错误 | 联系系统管理员查看系统日志，重启系统服务 |
| 报表生成失败 | 数据量过大或报表设计错误 | 优化报表设计，分批处理数据 |
